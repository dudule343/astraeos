Espace PRIVEOS Capital — Propriétaire de marque

Détail exhaustif de toutes les pages de l'espace du propriétaire de marque qui pilote un réseau de cabinets licenciés. PRIVEOS Capital est le 1er locataire d'ASTRAEOS de ce type. Cet espace est réservé aux tenants de type "Marque" — il n'est PAS activé pour les tenants "Cabinet direct" (cabinets indépendants qui achètent ASTRAEOS directement). Cet espace sert aussi de modèle de référence pour les futurs propriétaires de marque (Dupont Patrimoine, Mont-Blanc Conseil…).

💼 Tenant Cabinet direct

Licencié
Un cabinet partenaire de PRIVEOS Capital, indépendant juridiquement (sa propre structure ORIAS). Représenté par son dirigeant dans le Doc 4.
Ingénieur patrimonial
Un collaborateur d'un cabinet licencié. Suit ses propres clients dans son espace dédié (Doc 5). Cloisonné aux clients qui lui sont attribués.
Client final
Le client patrimonial accompagné. Géré par un licencié, mais visible dans le CRM réseau de PRIVEOS Capital. Possède son propre espace (Doc 6).
Captation directe
Prospect/client entré directement chez PRIVEOS Capital sans passer par un licencié. Peut être réattribué à un licencié ensuite.
Réseau
L'ensemble des cabinets licenciés + leurs ingénieurs + leurs clients. PRIVEOS Capital pilote le tout via cet espace.

⚖ Périmètre du document · Tenants Marque uniquement

Cet espace est réservé aux tenants de type "Marque" — c'est-à-dire les propriétaires d'un réseau de cabinets en franchise/licence (PRIVEOS Capital, et demain Dupont Patrimoine, Mont-Blanc Conseil…).

Pour les tenants de type "Cabinet direct" (cabinets indépendants qui achètent ASTRAEOS sans appartenir à un réseau · ~3 000 CGP en France) :

  • Le Doc 3 n'est PAS activé — ils n'ont pas de niveau "propriétaire de marque" au-dessus d'eux.
  • Leur architecture se simplifie en 3 niveaux : ASTRAEOS → Cabinet → Ingénieurs → Clients.
  • Ils utilisent uniquement les Doc 4 (Cabinet) + Doc 5 (Ingénieur) + Doc 6 (Client).
  • Leur catalogue produits, référentiels et outils sont gérés directement par le cabinet ou via la Marketplace ASTRAEOS (cf. Doc 2 section 4).

Voir Doc 1 Master · section "Les 2 modèles commerciaux d'ASTRAEOS" pour le détail complet.

  • ✅ Consultation de tous les dossiers du réseau
  • ✅ Édition directe possible sur les fiches clients (corrections, ajouts d'informations)
  • ✅ Actions de pilotage : réassigner dossier, déclencher revue compliance, escalader, mettre en pause

L'idée n'est pas de microgérer mais de pouvoir intervenir en cas de problème (ingénieur absent, dossier bloqué, alerte compliance, etc.).
